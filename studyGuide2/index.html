System On a Chip Architecture

Hover over the modern SOC to see where elements came from.

Performance Central Processing Unit Cores

The first product to have a defined core was the Intel 4004, released on November 15, 1971.

The CPU core design was developed as a more effective way for Intel to solve customer problems by building a more general purpose computational unit.

All modern computers utilize the transistor based core design pioneered by the Intel 4004.

Efficiency Central Processing Unit Cores

A recent development in computer design, efficiency cores are electrically efficient processing units to be used when the computational load is low.

Some modern CPU architectures will include a few less powerful Efficiency Cores as well as the normal Performance Cores.

These smaller circuits are designed in similar manner to P Cores but are not required to be as performant and instead attempt to improve thermal and electrical efficiency.

Modern GPUs utilize parallel processing to complete many numerical operations independently at the same time.

Neural Engine

This hardware accelerator is specifically designed to increase performance of machine learning training and evaluation.

These circuits can only do one task but are generally faster than general purpose compute units at their task.

When computers were first being developed, all of the circuits were problem specific and took hardware changes to reprogram.

With the rise of general purpose computing, problem specific electronics fell out of favor.

Recently hardware accelerators have become more popular to attempt to speed up common actions or complete actions that are not efficient for general purpose systems.

Memory Cache

First developed by Maurice Wilkes in 1965.

CPUs need instructions on what to do and data on which to execute those instructions.

All of this information is fed from the computer's memory.

Memory and storage is often slower than how fast a CPU can process some instructions.

For that reason, it is best to have a system with a small amount of fast access memory that can quickly feed the CPU with information without having to wait for storage access times.

Memory

The concept of memory has been important to any form of general computer even since the Analytical Engine.

Memory allows the computer to hold data and instructions to process.

Memory originated in the form of mechanical devices, then drum memory, then woven core memory. Eventually engineers developed vacuum tube memory and finally transistor based memory.

Input / Output

These circuits are responsible for communication between the SOC and the rest of the devices connected to the computer.
