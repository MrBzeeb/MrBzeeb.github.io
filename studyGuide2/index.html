System On a Chip Architecture

Hover over the modern SOC to see where elements came from.

Performance Central Processing Unit Cores

The first product to have a well defined CPU core was the Intel 4004, released on November 15, 1971.

The CPU core design was developed as a more effective way for Intel to solve customer problems by building a more general purpose computational unit.

Computer designers focused on integrating the important elements for computation onto one chip that could easily be repurposed to complete many different tasks.

The development of the Intel 4004 allowed people to use their computers to solve more varied problems without needing more excessively large systems.

All modern computers utilize the transistor based core design pioneered by the Intel 4004.

Efficiency Central Processing Unit Cores

A recent development in computer design, Efficiency Cores are electrically efficient processing units to be used when the computational load is low.

Some modern CPU architectures will include a few less powerful, more thermally and electrically efficient E Cores as well as the normal P Cores.

E Cores are part of manufacturers response to the ever growing power required to satisfy customer performance desires.

Computers and computing related technology is expected to draw 21% of the world’s electricity in 2030 and is likely to continue trending upwards.

Increased energy demands cause concerns for the sustainability of computing as the world slowly attempts to change to greener energy production.

Graphics Processing Unit

Originally called the Graphics Processor Unit and was built for 1970’s video games to handle instructions related to sending the video signal for the game.

Eventually the term GPU was redefined to mean Graphics Processing Unit which was largely popularized by Nvidia and the GeForce 256.

Modern GPUs utilize parallel processing to complete many numerical operations, independently, at the same time.

GPUs are the easiest entry into largely parallelized processing, a technique important to pop-culture with animation, video, and video games, as well as scientific or data science processing capability.

Neural Engine

This hardware accelerator is specifically designed to increase performance of machine learning based algorithms.

These circuits can only do one task but are generally faster than general purpose compute units at their task.

When computers were first being developed, all of the circuits were problem specific and took hardware changes to reprogram, but this architecture fell out of favor for the much easier to reuse general purpose computational design.

Recently hardware accelerators have become more popular to attempt to speed up common actions or complete actions that are not efficient for general purpose systems.

The Neural Engine and other like-minded machine learning hardware accelerators allow algorithm developers to run and train much more complex neural networks.

Memory Cache

Computer Memory Cache was first conceptualized by Maurice Wilkes in 1965.

Wilkes realized that CPUs were waiting on instructions or data to be returned from regular memory.

Overall system efficiency could be improved by adding a small amount of fast access memory that could hold batches of information for upcoming CPU tasks.

The development of cache is a strong example of how engineers manipulate computer architecture in attempts to improve desired results.

However, augmenting the architecture can cause vulnerability and biases like speculative execution bugs.

Memory

The concept of memory has been important to any form of general computer, even since the Analytical Engine.

Memory allows the computer to hold data and instructions to process.

Memory originated in the form of mechanical devices, then drum memory, then core rope memory. Eventually engineers developed vacuum tube memory and finally transistor based memory.

It has been extremely important to engineers to improve memory stability, compactness, speed, and capacity.

Improving memory is important to allow larger computations to happen more efficiently, whether for war-time calculations, Apollo guidance computers, or data science applications.

Input / Output

These circuits are responsible for communication between the SOC and the rest of the devices connected to the computer.

Humans cannot interact with the electrons moving around on a computer die, there must be some interface that allows interactivity with the system.

Computer peripherals allow for different devices to allow humans to access the processing power of a computer.

However, not all peripherals are made the same. The most common and effective devices might not be effective for people with certain concerns or disabilities.

A device that works to bring computers to people can also effectively push certain groups away due to interfacing issues.
